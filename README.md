🏦 Projeto Caixa Eletrônico (Terminal)
Este projeto é o resultado final dos meus estudos iniciais em Python. Ele simula o funcionamento de um caixa eletrônico diretamente pelo terminal, focando na lógica de programação e na estrutura de dados básica.

🚀 Funcionalidades
O sistema permite a interação com contas pré-definidas através de número de conta e senha, oferecendo as seguintes opções:

Saldo: Consulta instantânea do valor disponível em conta.

Saque: Retirada de valores (com atualização automática do saldo).

Depósito: Adição de valores fictícios à conta.

Extrato: Histórico detalhado de todas as movimentações realizadas.

Sair: Encerramento seguro da sessão no terminal.

🧠 O que eu pratiquei neste projeto:
Durante o desenvolvimento, apliquei conceitos fundamentais de programação que estou aprimorando:

Variáveis e Tipos de Dados: Manipulação de valores numéricos e textos.

Condicionais (If/Else): Regras de negócio para saques e acesso.

Dicionários: Organização dos dados das contas e usuários.

Funções: Modularização do código para deixá-lo mais limpo e reutilizável.

Estruturas de Repetição: Manutenção do menu ativo até que o usuário decida sair.

📚 Aprendizado
Este programa foi construído como um exercício de "aprendizado guiado". Utilize recursos como tutoriais no YouTube e o apoio de IAs (Gemini e ChatGPT) para entender as melhores práticas e resolver desafios de lógica.

Nota: Todos os valores e contas são fictícios, criados exclusivamente para fins de estudo e prática de programação.
